# Sistema de Gestión de Contactos

Aplicación web desarrollada siguiendo un patrón arquitectónico MVC estricto.

## Stack Tecnológico

- **Runtime:** Node.js 24 LTS
- **Lenguaje:** TypeScript
- **Framework:** Express
- **Template Engine:** Handlebars (HBS)
- **Base de Datos:** SQLite
- **Validación:** Zod

## Estructura de Directorios

```text
├── src/
│   ├── config/
│   ├── controllers/
│   ├── middlewares/
│   ├── models/
│   ├── routes/
│   ├── schemas/
│   └── views/
│       ├── layouts/
│       └── partials/
├── database/
└── ...
```

## Ejecución

Para iniciar el proyecto en modo desarrollo, ejecuta:

```bash
pnpm run dev
```
